This is my third "bare metal" project for the Rockchip RK3328.

The idea here is to accomplish these goals:

* Add a printf via prf.c from my RK3399 repo
* move uart code to uart.c
* move gpio code to gpio.c
* generalize gpio code as much as possible
* blink all 3 onboard LED
